cbourree_IUT
============

Je met ici toutes les fonctions que je créer dans le cadre de ma formation GEII IUT Gratte Ciel ^^

Les fonctions qui commencent par fu sont des fonctions demander par l'IUT
Les fonctions qui commencent par ft sont des fonctions que j'ai coder/recoder pour les optimiser/le fun, très souvant réutilisable
Les fonctions qui commencent par fs sont des sous fonctions utiles en générale uniquement pour le fonctionement d'un programme/une fonction.
